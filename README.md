## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Git e Github
- Figma
- Visual Studio Code

## 💻 Projeto

Este é um projeto de front-end desenvolvido para apresentar receitas de forma organizada e atrativa. O projeto visa fornecer uma interface intuitiva para os usuários visualizarem e acessarem diferentes receitas.

## 🔖 Layout

Você pode visualizar o layout do projeto através [DESSE LINK](https://www.figma.com/community/file/1360315130061454535/pagina-de-receita). É necessário ter conta no [Figma](https://figma.com) para acessá-lo.
